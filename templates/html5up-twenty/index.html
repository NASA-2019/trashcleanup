Re-Ocean Clean Boat

El equipo Re-Ocean está desarrollando un sistema dinámico para el challeges nuestros oceanos de NASA, utilizando las fuerzas oceánicas naturales para atrapar y concentrar el plástico. Tanto el plástico como el sistema están siendo transportados por el viento, las olas y la corriente. Sin embargo, para atrapar plásticos debe haber una diferencia constante en la velocidad entre el sistema y los plásticos. Usando un ancla de superficie para crear arrastre, el sistema se mueve más lento que el plástico, permitiendo que el plástico sea capturado.

Descripción del sistema

Nuestro dispositivo empaqueta en mallas con una capacidad de 130 toneladas la basura dispersa en el oceano y le enlaza un gps para poder rastrearla en el mar. El objetivo es que embarcaciones que transitan por el lugar puedan recuperar esto paquetes y tasladarlos hasta la orilla para que sean reciclados y convertirlos en productos útiles y de larga duración
